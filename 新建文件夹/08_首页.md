# 八、首页



###1.在 SwiperController 文件中新增拦截
```
@RestController
@RequestMapping("/swiper")
public class SwiperController {

    @Autowired
    SwiperServiceImpl swiperService;

    @GetMapping
    public WebResult getSwiper(){
        List<Swiper> list = swiperService.list();
        return WebResult.success(list);
    }
}
```
